# Assets Directory

This directory is for storing website assets such as images, icons, and other media files.

## Recommended Structure

```
assets/
├── images/
│   ├── products/          # Product images
│   ├── applications/      # Application area images
│   ├── factory/           # Factory photos
│   └── hero/              # Hero banner images
├── icons/                 # Icon files (SVG, PNG)
└── documents/             # PDF catalogs, datasheets, etc.
```

## Image Requirements

- **Product Images**: Recommended size 600x400px or higher
- **Hero Images**: Recommended size 1920x600px or higher
- **Format**: JPG, PNG, or WebP
- **Optimization**: Compress images before uploading for better performance
